# 🏆 Boss Raid: Hall of Fame (book)

보스 레이드 게임의 전설적인 기록들을 실시간으로 확인할 수 있는 **명예의 전당 포털**입니다.

## 🚀 주요 기능
*   **실시간 리더보드**: 게임(`book2`)에서 기록된 모든 승리 소식을 실시간으로 불러와 표시합니다.
*   **파이어베이스 연동**: 동일한 구글 클라우드 데이터베이스(Firestore)를 공유하여 데이터 무결성을 유지합니다.
*   **반응형 완벽 지원**: 데스크탑과 모바일 어디서든 깔끔하게 최신 기록을 확인할 수 있습니다.
